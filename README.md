# US Hospital Readmissions Analysis

**Tools:** Python | SQL | SQLite | Tableau Public
**Data:** CMS Medicare Hospital Readmissions (8,037 records, 51 states)

## Overview
Analyzed 30-day hospital readmission patterns across 3,000+ US hospitals to identify which conditions, states, and hospital sizes have the highest readmission risk.

## Key Findings
- Hip & Knee Replacement has the highest readmission ratio (1.036)
- Massachusetts is the highest risk state -- Idaho is the lowest
- Larger hospitals consistently outperform smaller ones
- 56% of records had insufficient data highlighting gaps in rural healthcare
